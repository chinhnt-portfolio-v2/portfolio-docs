## What Went Well

1. **Code Velocity** - Story implementation was fast (same day completion for most stories)

## What Didn't Go Well

1. **Deploy Pipeline Issues**
   - Multiple failed deployments due to various issues:
     - Java version mismatch (Java 23 vs Java 21)
     - Bucket4j causing container startup failure
     - Database column type mismatch (INET vs VARCHAR)
     - Flyway migration checksum conflicts
     - Response body mismatch (201 with no body)

2. **Problem Diagnosis**
   - Initial problem identification was inaccurate
   - Assumed infrastructure issue when it was code-related
   - Took entire day to debug and fix

3. **Code Quality Issues**
   - Contact form does not support Vietnamese language
   - Missing spacing between labels and inputs (poor UX)
   - Backend response missing JSON body caused frontend parsing error

## Action Items

1. **Add dedicated Testing Story** - Future epics should include separate story for testing (unit + integration + E2E)
2. **Add dedicated Deploy Story** - Future epics should include separate story for deployment verification
3. **Pre-commit Local Build Check** - Verify build passes locally before pushing to CI/CD
4. **Contact Form i18n** - Add Vietnamese support for contact form labels and messages
5. **Contact Form UX** - Add proper spacing between labels and inputs
